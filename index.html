Starbucks Drink Nutrition Explorer

Explore sugar vs calories for Starbucks drinks. Use the controls to filter and see how different drink categories compare.

Project documentation

How to read this chart

  • Each dot is a drink size + preparation.
  • The x‑axis shows calories; the y‑axis shows sugar (g).
  • Use the dropdowns to switch category and color encoding.
  • Hover over a dot to see the exact drink name and nutrition info.

Data source: Kaggle – Starbucks drink menu nutrition.
